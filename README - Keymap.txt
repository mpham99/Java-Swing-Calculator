Here is the following key map for the special characters

Button		Designate Key

Hex		Alt + H
.0		Alt + J
.00		Alt + K	
Sci		Alt + L
Clear		Delete
Plus/Minus	Alt + M

NOTE: Special case - For the equal '=' , both the '=' on the keyboard and the Enter button are usable
      Hex buttons (A-F) is case insensitive. Example: Press 'A' on the keyboard or Shift + A perform the same task.
